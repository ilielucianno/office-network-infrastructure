## Training Topics Covered

### 1. Phishing Awareness

**What employees learn:**
- How to identify suspicious emails
- Never click links from unknown senders
- Never download attachments unless verified
- Check sender email address (not just display name)
- Look for spelling errors, urgent language, fake invoices

**Real example discussed:** The crypto phishing incident where a user clicked a malicious link. Result: Funds were moved immediately, accounts reset, training reinforced.

**Red flags to watch for:**

| Red Flag | Example |
|----------|---------|
| Urgent action required | "Your account will be closed in 24 hours" |
| Suspicious sender | `admin@paypa1.com` instead of `admin@paypal.com` |
| Request for password | Legitimate companies never ask for passwords |
| Unexpected attachment | Invoice, shipping notice, voicemail .zip |

---

### 2. Password Security

**Rules enforced:**

| Rule | Explanation |
|------|-------------|
| Minimum 12 characters | Longer passwords are harder to crack |
| Use a password manager | Bitwarden, 1Password, or built-in browser manager |
| Never reuse passwords | Each account gets unique password |
| Never share passwords | Not via email, chat, or phone |
| Change every 6 months | Or immediately after any incident |

**What we do when a password is leaked:**
1. Detect via Google breach alert
2. Change password immediately
3. Reset 2FA
4. Review recent account activity

---

### 3. 2FA (Two-Factor Authentication)

**Mandatory for:**
- Zoho accounts (all users)
- Odoo (HR and Accounting)

**How it works:**
- Password + 6-digit code from authenticator app
- Without the code, login is blocked
- Backup codes saved for emergency

**What to do if phone is lost:**
- Notify IT immediately
- Use backup codes to log in
- Reconfigure 2FA on new device

---

### 4. Physical Security

**Rules:**
- Lock laptop when away from desk (`Win + L` or `Ctrl + Cmd + Q`)
- Never leave printed sensitive documents unattended
- Shred paper documents before discarding
- Report lost or stolen devices immediately

---

### 5. Data Protection (GDPR Basics)

**What employees must know:**
- Only access data needed for your role
- Do not download customer data to personal devices
- Do not share screenshots of sensitive data
- Report any suspected data breach immediately

**Incident reporting:** Contact Ilie Lucian immediately (Signal or phone)

---

### 6. Remote Work Security (Support Agents)

**Rules for remote staff:**
- Use company laptop only
- Connect only via WireGuard VPN
- Do not use public WiFi for work
- Keep VPN connected during work hours
- Report lost or stolen laptop immediately

**Restrictions:**
- Remote agents cannot access HR or Accounting systems
- Remote agents cannot approve withdrawals
- Remote agents have limited Zoho permissions

---

### 7. What to Do in Case of Incident

**Emergency contact:** Ilie Lucian

**Steps:**
1. Do not panic
2. Do not delete anything
3. Take screenshots if possible
4. Report immediately
5. Follow instructions

**Examples of incidents to report:**
- Clicked a suspicious link
- Entered password on fake site
- Lost phone with authenticator app
- Suspicious email received
- Cannot log in to account
